Music Library Web Application
Project Description

The Music Library Web Application is a full-stack web project developed using the MERN stack (MongoDB, Express.js, React.js, and Node.js).
This application allows users to manage and explore a collection of songs in a digital music library.

Users can view songs, search songs, filter songs by genre or language, mark songs as favourite, and see recommended songs.

Features
1. Mark Song as Favourite

Users can mark any song as their favourite.
Favourite songs can be easily identified and managed.

2. Recommended Songs

The application displays recommended songs in a separate section to help users discover music easily.

3. Genre-wise Song List

Songs can be viewed based on their genre (category), which helps users browse music according to their interest.

4. Language-wise Song List

Users can filter songs based on language such as Hindi, English, etc.

5. Search Songs

Users can search songs using:

Song Title

Artist Name

Album Name

6. Sort Songs by Release Year

Songs can be sorted based on release year from newest to oldest, making it easier to find the latest songs.
